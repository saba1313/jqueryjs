If you use cursorAt, you can use the wrapHelper option to always see the helper, even when coming to a edge of the screen. This is enabled by default. The second example shows it with option not enabled.

$("#mydiv").drag({
cursorAt: { top: -10, left: -10 }
});
Drag me
$("#mydiv2").drag({
cursorAt: { top: -10, left: -10 },
wrapHelper: false
});
Drag me
