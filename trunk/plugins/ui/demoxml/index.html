This is the basic drag 'n drop. Notice how easy it is to do this! At the moment, dropping does nothing. We'll get to that in the next example.
HTML:
           <div id='drag-1'>             Drag me!           </div>           <div id='drop-1'>             Drop on me           </div>
JavaScript:
           $('#drag-1').draggable();           $('#drop-1').droppable();

Let's make the droppable fade out when we drop something on it.
The way to do that is adding a drop function to the options.
HTML:
 
          <div id='drag-2'> 
            Drag me! 
          </div> 
          <div id='drop-2'> 
            Drop on me 
          </div>
JavaScript:
 
          $('#drag-2').draggable(); 
          $('#drop-2').droppable({ 
            drop: function() { 
              $('#drop-2').fadeOut('slow').fadeIn('fast'); 
            } 
          });

Resizables allow any item to be resized. A resizable also needs handles. However, basic ones don't. Note: the extra spans need to be dealt with. They must go.
HTML:
 
        <div id='resizable-1' style='width: 200px; height: 200px'> 
          <div class='ui-resizable-n ui-resizable-handle'><span style='display: none'>.</span></div> 
          <div class='ui-resizable-e ui-resizable-handle'><span style='display: none'>.</span></div>     
          <div class='ui-resizable-s ui-resizable-handle'><span style='display: none'>.</span></div> 
          <div class='ui-resizable-w ui-resizable-handle'><span style='display: none'>.</span></div> 
 
          <div class='ui-resizable-se ui-resizable-handle'><span style='display: none'>.</span></div> 
          <div style='width: 10px; height: 10px; bottom: 0px; left: 0px;' class='ui-resizable-sw ui-resizable-handle'><span style='display: none'>.</span></div> 
          <div style='width: 10px; height: 10px; top: 0px; right: 0px;' class='ui-resizable-ne ui-resizable-handle'><span style='display: none'>.</span></div> 
          <div style='width: 10px; height: 10px; top: 0px; left: 0px;' class='ui-resizable-nw ui-resizable-handle'><span style='display: none'>.</span></div> 
 
          <div style='position: absolute; top: 20px; left: 20px; bottom: 20px; right: 20px;'>Resize me!</div> 
        </div>
JavaScript:
 
        $('#resizable-1').resizable();
